# Behind The Scene Collective — Website

Everything you need is in this folder:
- `index.html` — the whole site
- `assets/logo.jpg` — your logo

## Before you deploy — fill these in

Open `index.html`, use Ctrl+F (or Cmd+F) to find and replace:
- `hello@yourdomain.com` → your real email
- `@behindthescenecollective` → your real Instagram/social handle
- `Add your city here` → your location (or delete that box if you'd rather not share it)

## Deploy for free with GitHub Pages

1. **Create a free GitHub account** at github.com if you don't have one.
2. **Create a new repository:**
   - Click the `+` in the top right → "New repository"
   - Name it anything, e.g. `bts-collective-site`
   - Set it to **Public**
   - Click "Create repository"
3. **Upload your files:**
   - On the new repo page, click "uploading an existing file"
   - Drag in `index.html` and the whole `assets` folder from this download
   - Click "Commit changes"
4. **Turn on GitHub Pages:**
   - Go to the repo's **Settings** tab → **Pages** (left sidebar)
   - Under "Build and deployment" → Source, select **Deploy from a branch**
   - Branch: `main`, folder: `/ (root)` → Save
5. **Wait ~1–2 minutes.** GitHub will show you a live URL like:
   `https://yourusername.github.io/bts-collective-site/`

That's it — completely free, no credit card, no expiration. Every time you upload a new version of `index.html`, the site updates automatically within a minute or two.
